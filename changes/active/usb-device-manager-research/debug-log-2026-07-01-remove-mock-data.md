# 删除演示假数据

## 背景

用户要求删除所有 Mock 假数据；没有真实数据时，应展示正确的空态提示页面，不再回填演示设备、演示文件或演示日志。

## 处理结果

- 前端删除静态演示设备、文件树和日志生成逻辑，初始化状态改为空真实状态。
- 后端删除演示日志服务和演示日志数据来源，旧演示日志路径只返回 404 错误，不返回任何日志内容。
- 后端不可达、无设备、无文件内容时，前端展示对应空态提示，不再展示演示内容。
- 新增前端资源控制器，显式返回打包进 jar 的首页和 Vite assets，避免静态资源异常影响页面加载。
